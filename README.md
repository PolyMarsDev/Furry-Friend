# Furry Friend

Furry Friend is a horror game made in 48 hours for [Ludwig Jam 2023](https://itch.io/jam/ludwig-2023).

## Background

You forgot to take your height-enhancing pills. Your pet cat, Coots, is hungry. Good luck.
